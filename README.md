# Mintlify Landing Page Clone

This project is a single-page landing website clone inspired by the Mintlify website, built using pure HTML and CSS.
It focuses on modern UI layout, hero sections, brand grids, feature sections, videos, cards, and a full footer layout.

# Live Demo:
https://mintlify-lyart.vercel.app/

# Technologies Used

HTML5

CSS3

GitHub Pages

GitHub Actions
